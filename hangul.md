# 🇰🇷 FICHE EXPRESS – APPRENDRE LE HANGUL

## 🔤 1. Voyelles (base essentielle)

```markdown
ㅏ = A → (→ droite)
ㅓ = EO → (← gauche)
ㅗ = O → (↑ haut)
ㅜ = OU → (↓ bas)
ㅣ = I → (│ ligne droite)
```

💡 **Mémo ultra rapide :**

* droite = A
* gauche = EO
* haut = O
* bas = OU

***

## 🔠 2. Consonnes principales

```markdown
ㄱ = g/k → angle (pistolet)
ㄴ = n → angle
ㄷ = d/t → porte
ㄹ = r/l → serpent
ㅁ = m → carré (bouche fermée)
ㅂ = b/p → carré + barre
ㅅ = s → dent
ㅇ = silencieux (début) / ng (fin)
ㅎ = h → souffle
```

***

## 🧩 3. Construction des syllabes

👉 1 bloc = 1 syllabe

```markdown
Consonne + Voyelle (+ Consonne)
```

### Exemples

```markdown
가 = ga
나 = na
무 = mou
한 = han
김 = gim
```

***

## 🔍 4. Lecture (règles simples)

```markdown
Lire : gauche → droite
       haut → bas
```

***

## ⚠️ 5. Règles IMPORTANTES

### ✅ ㅇ (cercle magique)

```markdown
Début = silencieux → 아 = a
Fin = ng → 방 = bang
```

***

### ✅ ㅅ + ㅣ = SH

```markdown
시 = shi
신 = shin
```

💡 Mémo :
“S + i → devient ‘ch’”

***

### ✅ ㄹ = R ou L

```markdown
Début = R → 라 = ra
Fin = L → 물 = mul
```
